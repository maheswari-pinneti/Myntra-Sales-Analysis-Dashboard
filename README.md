# Myntra-Sales-Analysis-Dashboard

## 🚀 Project Overview

The **Myntra Sales Analysis Dashboard** is a Power BI project designed to analyze sales performance across multiple product categories, brands, states, and years. The dashboard transforms raw retail data into meaningful business insights through interactive visualizations, KPI tracking, and trend analysis.

This project demonstrates how Business Intelligence can help organizations monitor performance, identify opportunities, and make data-driven decisions.

---

## 🎯 Business Objective

To analyze Myntra sales data and provide actionable insights regarding:

* Revenue Performance
* Sales Trends
* Customer Purchasing Behavior
* Brand Performance
* Product Performance
* Regional Sales Distribution
* Discount Effectiveness

---

## 📌 Key Features

### KPI Dashboard

* Total Revenue

### Interactive Filters

* Year
* Category

### Advanced Analytics

* Top-Selling Brand Analysis
* Monthly Sales vs Discount Analysis
